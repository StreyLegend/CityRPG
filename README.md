# CityRPG
 <h2>Описание</h2>
<p>Игрок оказывается на необитаемом острове. Из-за крушений корабля. Он практически все забыл. Главный герой возвращается на порт с помощью его капитана и хочет главному герою помочь, он даёт первые деньги на проживание и говорит, где можно ещё подзаработать. Перед игроком открывается жесткий, необычный мир. Ему придется, начать с дряхлого меча. Но, ты в этом ему поможешь.</p>
